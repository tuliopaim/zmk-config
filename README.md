## Features

- **Home Row Mods**: Modifiers on home row (ASDF / JKL;) for ergonomic access
- **Mouse Control**: Layer 2 includes mouse movement and scroll wheel controls
- **Function Keys**: Layer 3 provides F1-F12 and system controls
- **Bluetooth**: Support for 3 Bluetooth profiles plus clear function
